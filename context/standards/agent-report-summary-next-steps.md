# Agent Report Summary and Next Steps

> Agents should always report summary and next steps at the end of their work.

## Rationale

Ensures that after any agent action (whether Vibuzo executing directly or Deepveloper reporting back from a subtask), the user is always informed of:
1. **What was accomplished** — a concise summary of work done
2. **What comes next** — clear next steps, action items, or decisions needed from the user
