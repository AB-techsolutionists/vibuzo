Review the implementation of feature: $1

1. Read `specs/$1/spec.md`, `specs/$1/plan.md`, `specs/$1/tasks.md`
2. Find the implemented code in the codebase
3. Compare against what was specified and planned

Generate a review report covering:
- **Coverage**: Does the implementation cover all requirements?
- **Accuracy**: Does it match the spec and plan?
- **Quality**: Does the code follow established principles?
- **Gaps**: What's missing or incomplete?
- **Issues**: Bugs, tech debt, or concerns found

Present the report with clear action items. Do not modify any files.
